## 桌面/Launchpad/开始菜单快捷方式生成器
- 需求：把系统启动做成快捷方式，放在桌面或程序抽屉(Launchpad)/应用列表(Windows)。
- 新增两个生成器脚本（运行一次即可在目标机器生成快捷方式）：
  - `scripts/create-shortcut.sh`（mac）：生成 `Amazon 经营工具箱.app`（启动）+ `-停止.app`（停止）标准 .app 包；项目路径在生成时写入 launcher；`LSUIElement=true`（不占 Dock）；启动后等待端口就绪→打开浏览器→osascript 通知；软链接到 `~/Applications`（Launchpad 可见）+ `~/Desktop`。
  - `scripts/create-shortcut.ps1`（windows）：用 WScript.Shell COM 创建 `.lnk`；开始菜单(启动+停止) + 桌面(启动)；launcher 等待端口→打开浏览器→MessageBox 提示。
- `.gitignore` 新增 `*.app/` 和 `scripts/launch-from-shortcut.ps1`（生成物不入库）。
- 实测（macOS bash 3.2）：启动 app → 服务启动 + /api/health=ok + 浏览器打开 + 通知；停止 app → 进程清理 + 端口释放 + 通知。
- 已 commit 并 push：`58b0d70`。
- 使用方式：`bash scripts/create-shortcut.sh`（mac）或 `powershell -File scripts/create-shortcut.ps1`（windows），每台机器各跑一次。
